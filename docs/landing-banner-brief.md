- **피할 것**: 과도한 3D, 너무 화려한 네온, B2C 광고 톤, 경쟁사 실명, 카카오 공식 로고 무단 모방

## 4. 제품 상세 (배너 일러스트·아이콘 아이디어)

### 디대봇
- **채널**: 카카오톡
- **기능**: 회의실 예약, 점심 메뉴, 사내 규정·복지, 그룹웨어 Q&A
- **비주얼 힌트**: 노란 배경, 말풍선, 챗봇, 🤖

### SEE:SIGN
- **정의**: 디지털대성 전용 내외부 데이터 플랫폼
- **연결 데이터**: 뉴스, GDN·메타·유튜브 광고, 디시, 네이버 카페, 오르비, 마이맥, 유튜브, 블로그, X
- **NEW**: 뉴스 모니터링(육하원칙·일간 레포트·키워드 알림), 광고 모니터링(전체 DB·스마트 아카이브·AI 인사이트)
- **비주얼 힌트**: `#5E72E4`, 데이터 허브·노드 연결, 대시보드, ✦ NEW 배지

### AiBle CHAT
- **브랜드**: AI + ABLE = **AiBle** (디지털대성 CREW가 만든 도구)
- **기능**: 엑셀 업로드 → AI 분석·차트·PDF (SEE:SIGN 엑셀과 연동)
- **비주얼 힌트**: 다크 배경, 보라·핑크 그라데이션, 스파클, 엑셀·차트

## 5. 배너 사이즈 권장

|------|-------------|------|
