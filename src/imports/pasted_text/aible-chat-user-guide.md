# AiBLE CHAT 사용자 가이드

**엑셀 데이터와 대화하는 AI 분석 도구**

---

## AiBLE CHAT이란?

**AiBLE CHAT**(SEE:SIGN CHAT)은 엑셀(.xlsx, .xls, .csv) 파일을 업로드한 뒤, **평소 보고서에 쓰듯 자연어로 질문**하면 AI가 데이터를 읽고 답변해 주는 서비스입니다.

표를 직접 피벗하거나, 수천 줄을 눈으로 훑지 않아도 됩니다.  
예를 들어 이렇게 물어볼 수 있습니다.

- 「이번 달 매출 추이를 차트로 보여줘」
- 「어느 교재 페이지에 질문이 가장 많아?」
- 「수험생들이 어떤 플랫폼을 많이 언급해?」
- 「P.129 2번 문항 질문만 정리해줘」

답변은 **표·요약·그래프** 형태로 정리되며, 필요하면 **PDF 저장·복사**도 할 수 있습니다.

---

## 이런 분께 잘 맞습니다

| 상황 | AiBLE CHAT이 도와주는 일 |
|------|--------------------------|
| **마케팅·기획** | 커뮤니티·후기에서 반복되는 니즈, 불만, 경쟁사 언급 파악 |
| **강사·운영** | Q&A에서 **질문이 몰린 교재·강의 구간** 찾기, 원인·액션 아이디어 정리 |
| **경영·매출** | 일자별·품목별 매출·건수 등 **수치 집계·추이** 확인 |
| **리서치** | 뉴스·기사·댓글 모음에서 주제·키워드 요약 |
| **보고 준비** | 여러 파일을 올려 놓고 대화하며 인사이트·초안 뽑기 |

**엑셀에 정리된 데이터가 있다**면, SEE:SIGN에서 뽑은 파일이든 다른 곳에서 받은 파일이든 대부분 활용할 수 있습니다.

---

## SEE:SIGN 데이터 — 특히 잘 맞는 유형

SEE:SIGN Admin에서 수집·보낸 데이터는 컬럼 구조가 잘 맞춰져 있어 **아래 유형에서 분석 품질이 특히 좋습니다.**

### 1. 커뮤니티·게시글 데이터

**예시 컬럼:** 게시판, 제목, 본문, 조회수, 추천수, 라벨-○○ 등

**잘 되는 질문 예시**

- 「주로 어떤 게시판·주제가 많아?」
- 「대성·메가 같은 플랫폼 언급이 어떻게 나와?」
- 「학생들이 자주 말하는 불만·칭찬 키워드 정리해줘」
- 「○○ 주제와 관련된 글 요약해줘」

긴 본문은 AI가 **질문과 관련된 글을 골라** 읽는 방식으로 분석합니다. 답변 아래에 **참고한 출처**가 표시될 수 있습니다.

---

### 2. Q&A(질문·답변) 데이터

**예시 컬럼:** 제목, 본문, 답변, 게시날짜,  
`질문 대상 (교재 위치)`, `질문 대상 (강의 영상 위치)` 등

**잘 되는 질문 예시**

- 「어느 **교재·페이지·문항**에 질문이 가장 많아?」 (핫스팟 순위)
- 「어느 **강의 차시·동영상 구간**에서 막혔어?」
- 「P.37 3번 문항 — 왜 질문이 많았는지 분석해줘」
- 「강사에게 전달할 인사이트 정리해줘」

Q&A는 **전체 행을 미리 집계한 핫스팟 리포트**를 우선 사용합니다.  
「몇 건인지」는 순위표 숫자를, 「왜인지」는 해당 위치의 **질문 본문 전체**를 바탕으로 분석합니다.

---

### 3. 매출·통계·수치 데이터

**예시 컬럼:** 일자, 매출일시, 공급가액, 결제 금액, 수량, 건수, 품목명 등

**잘 되는 질문 예시**

- 「공급가액 기준 **일자별 매출 트렌드** 분석해. 차트도 같이」
- 「품목별 매출 상위 10개」
- 「이번 기간 합계·평균 알려줘」
- 「월별 건수 비교 표로」

수치·매출·추이 질문은 **전체 행을 서버에서 집계**한 뒤 답합니다. (일부 행만 샘플링하는 방식이 아닙니다.)

> **참고:** 행 수가 매우 많은 파일(수천~만 행)은 **일자별·월별 요약** 질문이 **행 단위 전체 표**보다 더 안정적입니다. 「트렌드」「차트」처럼 **기간·집계 단위**를 명시해 주세요.

---

### 4. 그 밖의 SEE:SIGN보내기

- **수강 후기·리뷰** (본문·평점·날짜)
- **뉴스·기사** 모음
- **통계 시트** (qna_stats, posts 등 파일명·시트명에 통계·매출 힌트가 있는 경우)

컬럼 이름과 내용을 보고 AI가 **정량(숫자)** 인지 **정성(글)** 인지 자동으로 구분해 분석 방식을 바꿉니다.

## SEE:SIGN이 아닌 엑셀도 사용할 수 있나요?

**네.** 아래 조건만 맞으면 됩니다.

| 조건 | 설명 |
|------|------|
| **파일 형식** | `.xlsx`, `.xls`, `.csv` |
| **1행 = 헤더** | 첫 줄에 컬럼 이름(일자, 금액, 제목, 본문 등) |
| **표 형태** | 행·열로 정리된 표 데이터 |

**예시**

- 사내 매출 엑셀, POS·ERP보내기
- 설문 결과, 고객 문의 목록
- 마케팅팀이 정리한 후기·댓글 시트

Q&A 전용 컬럼(교재 위치, 강의 위치)이 없어도 **일반 표·텍스트 분석**은 가능합니다.  
다만 **교재 페이지 핫스팟**처럼 SEE:SIGN Q&A 형식에 맞춘 기능은 해당 컬럼이 있을 때 가장 정확합니다.

## 데이터는 어디서 구하나요?

파일이 없다면 화면 왼쪽 업로드 영역 아래 **「SEE:SIGN으로 데이터 수집하러 가기」** 버튼을 눌러  
[SEE:SIGN Admin](https://seesign-admin.digitalds.store/main)에서 매출·커뮤니티·Q&A 등을 수집·보낸 뒤, 받은 엑셀을 AiBLE CHAT에 올리면 됩니다.

---

## 사용 방법 (5단계)

### 1단계. 파일 업로드

1. 왼쪽 **「대화할 파일 업로드」** 영역에 엑셀을 **드래그**하거나 **클릭**해 선택합니다.
2. **한 번에 최대 5개**까지 올릴 수 있습니다.
3. 업로드되면 파일명, 시트 수, 행 수가 목록에 표시됩니다.

### 2단계. 학습 대기

업로드 직후 **「SEE:SIGN이 파일을 학습 중입니다」** 진행 표시가 나올 수 있습니다.  
텍스트가 많은 파일(게시글·Q&A 등)은 잠시 걸릴 수 있습니다. **「학습 완료」** 후 질문하는 것이 좋습니다.

### 3단계. 질문하기

오른쪽 채팅창에 **하고 싶은 분석을 문장으로** 입력합니다.

- 좋은 예: 「공급가액 기준 일자별 매출 추이 + 차트」
- 좋은 예: 「교재 핫스팟 TOP 10 표로」
- 좋은 예: 「○○ 키워드 언급 많은 글 주제 요약」

분석 중에는 **진행 단계**(파일 확인 → 리포트 생성 → 답변 작성 등)가 표시될 수 있습니다.

### 4단계. 답변 활용

- **표·불릿·강조**로 정리된 답변을 읽습니다.
- **차트**를 요청하면 막대·선·원형·히트맵 등이 화면에 그려질 수 있습니다.
- 답변 옆 **복사·PDF 저장**으로 보고서·공유용으로 남길 수 있습니다.
- 이어서 질문할 수 있으며, **「새 대화」**로 맥락을 초기화할 수도 있습니다.

### 5단계. 파일 추가·삭제

- 같은 대화에 **파일을 더 올리거나**, 목록에서 **X**로 제거할 수 있습니다.
- **「전체 삭제」**로 업로드 목록을 비울 수 있습니다.

---

## 질문 잘하는 팁

1. **목적을 한 문장에** — 「매출 추이」「핫스팟」「왜 많은지」처럼 의도를 분명히.
2. **기준 컬럼·기간** — 「공급가액 기준」「일자별」「3월만」 등.
3. **출력 형식** — 「표로」「차트로」「TOP 10」「강사용 인사이트로」.
4. **여러 파일** — 「A파일과 B파일 매출 비교」처럼 파일 역할을 말해 주면 좋습니다.
5. **Q&A 건수** — 「몇 건」은 **핫스팟 순위표**를, 「왜」는 **원인 분석**을 요청하세요.

---

## 차트·시각화

「차트」「그래프」「시각화」를 요청하면 답변에 **그래프가 함께** 나올 수 있습니다.

| 유형 | 언제 쓰면 좋은지 |
|------|------------------|
| 막대·가로 막대 | 항목별 건수·금액 비교 |
| 선 그래프·영역 | **일자·월별 추이** |
| 원형·도넛 | 비율·구성 |
| 히트맵 | 요일×시간, 페이지×문항 **밀도** |
| 순위 변동(bump) | 기간별 순위 변화 |
| 포지셔닝 맵 | 2축(가격 vs 품질 등) 배치 |
| 퍼널 | 단계별 전환·건수 |

## 알아두면 좋은 점 (한계·주의)

| 항목 | 내용 |
|------|------|
| **파일 개수** | 최대 **5개** |
| **파일 크기** | 너무 큰 파일은 업로드·분석이 실패할 수 있습니다. 가능하면 기간·항목별로 나눠 올려 주세요. |
| **대용량 매출** | 수천~만 행은 **「일자별 합계」「월별 트렌드」**처럼 집계 단위를 지정하는 것이 좋습니다. |
| **정확도** | 답변은 **업로드한 데이터만** 근거로 합니다. 파일에 없는 내용은 추측하지 않습니다. |
| **개인정보** | 민감한 개인정보가 포함된 파일은 업로드 전 **마스킹·삭제**를 권장합니다. |
| **새 대화** | 주제를 바꿀 때 **「새 대화」**를 누르면 이전 질문 맥락이 초기화됩니다. |

오류가 나면 **파일을 다시 업로드**하거나, **파일 수·크기를 줄인 뒤** 같은 질문을 다시 시도해 보세요.

## 자주 묻는 질문

**Q. 코딩·엑셀 함수를 알아야 하나요?**  
아니요. 일상 말로 질문하면 됩니다.

**Q. 여러 시트가 있는 파일은?**  
파일 안의 **시트를 모두 읽습니다.** 질문할 때 「○○ 시트만」이라고 적어 주면 더 정확합니다.

**Q. 학습 실패가 뜨면?**  
파일을 제거한 뒤 다시 업로드하거나, 파일 크기를 줄여 보세요.

**Q. SEE:SIGN과 AiBLE CHAT 차이는?**  
**SEE:SIGN Admin**에서 데이터를 **수집·관리**하고, **AiBLE CHAT**에서 그 데이터로 **질문·분석**합니다.

**Q. 답변을 팀에 공유하려면?**  
답변의 **PDF 저장** 또는 **복사** 기능을 사용하세요.
